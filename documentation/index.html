Getting Started

Welcome to NexaAI 👋

Thank you for purchasing the NexaAI - AI Support Agents SaaS HTML Template. This documentation will help you understand the code structure and how to customize the theme for your project.

Prerequisites

You need basic knowledge of HTML, CSS/SCSS, and JavaScript to fully customize this template.

Installation & Setup

NexaAI uses SCSS for styling to make customization incredibly fast and easy. To compile the SCSS files into standard CSS, you will need to compile them.

Compiling SCSS to CSS

If you have Node.js and Sass installed globally on your machine, you can run the following command in your terminal from the root folder of the template:

Terminal 
# Install Sass globally (if you haven't already)
npm install -g sass

# Watch and compile SCSS to CSS
sass --watch assets/scss/style.scss:assets/css/style.css

CSS & SCSS Architecture

NexaAI is built using modern SCSS with a modular approach. All variables (colors, fonts, spacing) are defined in _variables.scss and mapped to CSS Custom Properties (Variables) in the :root selector.

How to change the Theme Colors?

Open assets/scss/_variables.scss and modify the core hex codes at the top of the file. The SCSS will automatically generate all gradients, glows, and hover states across the entire template.

_variables.scss 
// Change these values to match your brand
$nx-primary:          #00D4FF; 
$nx-secondary:        #6C63FF;
$nx-accent:           #39FF14;

JavaScript (app.js)

All custom interactive features are handled in assets/js/app.js. The code is heavily commented into clear numbered sections.

Customizing Chatbot Responses

To change what the demo chatbot replies with, find Section 7 in app.js and edit the aiResponses array:

const aiResponses = [
  "Integrating NexaAI takes under 60 seconds. Want the guide?",
  "Our Starter plan is $49/mo, with 1,000 resolutions included.",
  "Your custom message goes here..."
];

General Customization

Changing Particle Colors

If you want to change the network background colors, go to app.js (Section 9) and update the hex codes in the tsParticles.load configuration.

Animations (GSAP & AOS)

Scroll animations are handled by AOS (edit data-aos attributes in HTML) and complex reveal animations are handled by GSAP ScrollTrigger in JS.
